Common Interview Questions & Concepts:

Stack ek linear data structure hai jo ek khaas order follow karta hai: LIFO (Last-In, First-Out). Iska matlab hai jo element sabse aakhir mein daala gaya (push kiya gaya), wohi sabse pehle nikala jayega (pop kiya jayega).

Imagine karo plates ka ek stack (dher). Aap nayi plate sabse upar rakhte ho (Push) aur jab plate nikalni hoti hai, toh bhi sabse upar wali hi nikalte ho (Pop). Aap beech mein se plate nahi nikal sakte.

Upar diagram mein 'C' sabse aakhir mein daala gaya tha, isliye 'Top' pointer usi ko point kar raha hai. Agar hum 'Pop' karenge, toh 'C' hi niklega. Agar 'Push' karenge, toh naya element 'C' ke upar aayega.

Stack ke primary operations hote hain:

  • Push: Stack ke 'Top' par ek naya element add karna.
  • Pop: Stack ke 'Top' se element ko remove karna aur (optional) return karna.
  • Peek (ya Top): Stack ke 'Top' element ko dekhna (bina remove kiye).
  • isEmpty: Check karna ki stack khali hai ya nahi.
  • isFull (Optional): Agar stack ka size fixed hai (e.g., array implementation), toh check karna ki stack full hai ya nahi.
  • Size: Stack mein kitne elements hain, yeh pata karna.

Time Complexity: Agar stack ko Array/List ya Linked List se implement kiya gaya ho, toh Push, Pop, Peek, isEmpty sabhi operations O(1) (Constant Time) mein ho jaate hain, jo ki bahut efficient hai!

Space Complexity: O(n), jahan 'n' stack mein store kiye gaye elements ki sankhya hai.

Stack ko do common tareekon se implement kiya jaa sakta hai:

  1. Using Array (or List in Python):
    • Ek array/list lein.
    • 'Top' ko track karne ke liye ek variable (index) rakhein. Initially -1 ya size 0.
    • Push: Element ko 'Top' ke agle index pe daalein aur 'Top' ko badhayein. Check karein ki array full toh nahi.
    • Pop: 'Top' index se element nikaalein aur 'Top' ko ghatayein. Check karein ki array khali toh nahi.
    • Peek: 'Top' index wale element ko return karein.
    • Python mein List ke built-in `append()` (push ke liye) aur `pop()` methods LIFO behaviour dete hain, isliye list se stack banana bahut aasan hai.
  2. Using Linked List:
    • Ek Singly Linked List use karein.
    • Push: Naye element ko list ke *beginning* (Head) mein add karein (O(1) operation). Head pointer naye node ko point karega.
    • Pop: List ke *beginning* (Head) se node ko remove karein (O(1) operation). Head pointer agle node ko point karega.
    • Peek: Head node ka data return karein.
    • Ismein stack 'overflow' (full hone) ki chinta nahi hoti (jab tak memory available hai).

Python mein List se simple implementation:


my_stack = [] # Ek khali list ko stack ki tarah use karenge

# Push operations
my_stack.append('A') # Stack: ['A']
my_stack.append('B') # Stack: ['A', 'B']
my_stack.append('C') # Stack: ['A', 'B', 'C']  <- Top is 'C'

print("Current Stack:", my_stack)

# Peek operation (last element)
top_element = my_stack[-1]
print("Top element (Peek):", top_element) # Output: C

# Pop operation
popped_element = my_stack.pop()
print("Popped element:", popped_element) # Output: C
print("Stack after pop:", my_stack) # Output: ['A', 'B']

popped_element = my_stack.pop()
print("Popped element:", popped_element) # Output: B
print("Stack after pop:", my_stack) # Output: ['A']

# isEmpty check
if not my_stack: # Check if list is empty
    print("Stack is empty")
else:
    print("Stack is not empty")

Stacks ke bahut saare practical applications hain:

  • Function Call Management: Programming languages function calls (aur unke local variables) ko manage karne ke liye 'Call Stack' ka use karti hain.
  • Undo/Redo Functionality: Text editors ya software mein 'Undo' operation implement karne ke liye (har action ko stack pe push karo, undo ke liye pop karo).
  • Expression Evaluation: Infix expression ko Postfix/Prefix mein convert karna ya Postfix/Prefix expression ko evaluate karna.
  • Parentheses Checking: Code mein brackets `{}`, parentheses `()`, square brackets `[]` sahi se balanced hain ya nahi, yeh check karna.
  • Backtracking Algorithms: Maze solving ya problems jahan explore karke wapas aana padta hai (e.g., Depth First Search - DFS implicitly stack use karta hai).
  • Browser History: Browser mein 'Back' button ka functionality (har visit ko stack pe push karo).
