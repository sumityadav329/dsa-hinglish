Common Interview Questions & Concepts:

Simple shabdon mein, Array ek collection hota hai similar type ke items ka jo memory mein ek saath (contiguous) store hote hain. Imagine karo ek building jisme consecutive numbered flats hain - yeh ek array jaisa hai. Har item ko uske 'index' (address ya flat number, jo 0 se start hota hai) se access kar sakte hain.

Upar diye gaye diagram mein `marks` naam ka ek array hai jisme 5 elements hain. Har element ka apna index hai (0 se 4 tak). Yeh sabhi elements memory mein ek ke baad ek store hote hain.

Example: `marks = [85, 92, 78, 90, 88]`. Yahan `marks[0]` 85 hai, `marks[1]` 92 hai, and so on.

Fayde: Kisi bhi element ko direct uske index se access karna bahut fast hota hai (O(1) time).

Nuksaan: Size fix hota hai (static arrays mein), naya element beech mein insert karna ya delete karna slow ho sakta hai kyonki baaki elements ko shift karna padta hai. Dynamic arrays (jaise Python lists ya Java ArrayList) size ka issue solve karte hain, lekin kabhi kabhi resize hone par performance hit ho sakti hai.

Python mein built-in 'list' hoti hai jo dynamic array ki tarah kaam karti hai. Yeh aise banti hai:


# Ek list (array) banayi jisme integers hain
my_marks = [90, 75, 88, 95, 60]

# Pehla element (index 0) print karna
print("Pehla student ke marks:", my_marks[0])  # Output: 90

Yaad rakho, Python mein list ka index 0 se shuru hota hai.
